# Phase 1 Foundation 개요

## 학습 구조
Foundation는 아래 3개 파트로 분리해 학습합니다.

1. 자료구조 (Data Structures)
2. 알고리즘 (Algorithms)
3. Java 기본기 (Java Core)

각 파트는 아래 순서로 진행합니다.
- 개념 이해
- Java 코드 예시 확인
- 파트 과제 수행

마지막에는 통합 과제(`assignment.md`)로 3개 파트를 한 번에 검증합니다.
