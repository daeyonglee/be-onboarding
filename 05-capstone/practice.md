# Phase 5 Practice — Capstone 준비 실습

## 실습 1) NFR 요구사항 도출
### 목표
기능 요구사항만이 아니라 성능/가용성/관측성 요구사항을 정의한다.

### 수행 항목
- 기능 요구사항 5개 작성
- NFR 5개 작성(성능/신뢰성/운영/보안)
- 각 항목별 측정 방법 명시

## 실습 2) 장애 시나리오 테이블 작성
### 목표
장애를 가정한 복구 전략을 사전에 설계한다.

### 템플릿
| 시나리오 | 영향 | 탐지 지표 | 자동 대응 | 수동 대응 |
|---|---|---|---|---|
| 결제 API timeout | 주문 지연 | timeout rate | 3회 재시도 | CS 공지 + 재처리 |
| 중복 이벤트 수신 | 재고 오염 | idempotency miss | eventId 차단 | 데이터 정합성 점검 |
